After Inserting elements 1, 32, 22, 15, 33, 45, 66, 73, 10. How many collisions occur when you hashCode() as key % size after inserting keys into hashtable for given sizes below.

Size = 7 
Size = 8
Size = 12
Size = 1

Answers:
Size = 7 	collisions=5
Size = 8	collisions=3
Size = 12	collisions=3
Size = 1	collisions=8
